# melonDS Installer Script

This repository provides a simple script to automate the installation and compilation of **melonDS** on Linux systems.

## Supported Distributions

- **Linux Mint 22.1 or newer**
- **Ubuntu (22.04 or newer)**

## Requirements

- A supported Linux distribution
- Internet connection
- sudo privileges

## Installation Instructions

Clone this repository:

```bash
git clone https://github.com/hudsonalbuquerque97-sys/MelonDS_installer.git

Enter the repository directory:

cd MelonDS_installer

Give execution permission to the script:

chmod +x install_melonds.sh

Run the installer:

./install_melonds.sh
````
The script will automatically install dependencies, compile melonDS, and set it up on your system.
